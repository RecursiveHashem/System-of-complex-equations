# System-of-complex-equations
Aims to solve a system of complex equations on a ti-83 python edition calculator (that doesn’t support the complex library).
With the additionnal option of being able to give a solution in terms of one or more variables.

FULL VERSION WITH ALL FEATURES REQUIREING THE LEAST AMOUNT OF RAM TO RUN IS INSIDE THE "ICantStop" FILE

Syntax tut: never use spaces
1 --> '1'
-2 --> '-2'
3.4 --> '3.4'
-5.6 --> '-5.6'

3+4j --> '3,4'

(6+7j)*Y --> '6,7,Y'
(6+7j)*Y + 8+9j --> '6,7,Y+8,9'
(you can use '+' any amount of times)
(6.2-7j)*Y + (8+9j)*z + 10j --> '6.2,-7,Y+8,9,z+0,10'
